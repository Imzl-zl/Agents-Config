## 项目记忆功能

本项目主要分享的是一个项目记忆功能，在vibe coding 的时候，重启会话，或者压缩之后，会丢失项目状态，又要重新开始和 AI 进行一番深入交流，太麻烦，然后有时候因为环境原因，使用命令报错，AI也记不住每次都要使用错误命令报错，之后才知道换正确的命令，太浪费上下文和时间，所以我搞了这个项目（使用自定义命令也是方便，因为之前我也不懂skills什么的安装都不会装）使用自定义命令会创建一下三个文件：

- **memory.md**（记录当前状态快照 + 最近活跃窗口）
- **tools.md**（稳定协作知识、常用命令、模式、坑点，自定义命令）
- **memory/archive/YYYY-MM.md**（按月追加的重要协作历史-这个是因为我在用的时候 memory.md 越来越大，上千行，我就又分出一个专门存放历史的）

## 使用方式

GitHub: https://github.com/Imzl-zl/Agents-Config

下载项目复制文件到用户目录的 `.claude/commands` 或者 `.codex/prompts` 中，然后：

- **claude code** 使用 `/init-project`
- **codex** 使用 `/prompts:init-project`

执行如果是新项目没有项目 `CLAUDE.md` 或者 `AGENTS.md` 执行之后会根据当前项目状态自动生成 `CLAUDE.md` 或 `AGENTS.md` 和 `memory.md`、`tools.md`、`memory/archive/YYYY-MM.md`，如果项目已有 `CLAUDE.md` 和 `AGENTS.md` 执行命令之后，会先备份你的文件（后悔了可以恢复），然后根据已有的文件内容进行关键规则提取和当前项目状态和最近8条 git 记录来生成 `CLAUDE.md` 或 `AGENTS.md` 和 `memory.md`、`memory/archive/YYYY-MM.md`
